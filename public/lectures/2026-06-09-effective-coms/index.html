Scenario A

You need to purchase two Linux servers to run workflows and manage data stored on a network attached file system. However, the lab has only ever used Windows environments and no one has ever asked for a "Linux" before.

Scenario B

The lab was able to start using cloud resources at the start of the year and you arrive at work one morning to find that the monthly costs are 1,000s of dollars more expensive than expected.
